How to use this tool:

Search IRs on DSxR&D Authoring with filter on IR family owner = g6c, g4k, jrz2, two1, kwa3, mwi5, iwk2, bza3
In field State select all values from 1 to 5.
Make Search, and then click on Tool icon to Export your search.
Open downloaded .csv file in text editor (like f.e. Notepad++)
Copy text from word "Incident" to the last word (without last comma)
Paste text to Textbox on this site and click Proceed.
